### CLone The Repo
    $ git clone  https://github.com/mahasagar/rating_system.git
    $ cd rating_system

### Install Dependency
    $ npm install
    $ bower install

### How to Run
    $ nodemon server.js   #Local

### How to Deploy
    $ pm2 start server.js  #Production


### Tech Stack
    - Angularjs : 1.6.5
    - Nodejs : 8.9.0
    - MongoDB : 3.4
